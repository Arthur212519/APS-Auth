# Auth API com JWT e Rate Limit

Projeto simples de estudo de autenticação usando Node.js, Express e JWT.

## Tecnologias
- Node.js
- Express
- JSON Web Token
- express-rate-limit
- dotenv

## Rotas
- POST /login
- GET /protected
- GET /private (admin)

Por enquanto, não implementei uma área de login e senha, todos os testes eu fiz utilizando insomnia testando cada rota.
